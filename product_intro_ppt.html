用户注册API服务

一站式用户注册与认证解决方案

v1.0.0

核心功能

  • POST /api/v1/register - 用户注册接口
  • GET /api/v1/user/{id} - 用户信息查询
  • API密钥管理 - 自动生成、安全存储
  • 健康检查 - GET /health 监控接口

技术架构

后端框架

Flask + SQLite

认证方式

API密钥 + SHA256密码哈希

跨域支持

Flask-CORS

部署方式

Docker / 本地运行

快速开始

# 启动服务
python api_server.py

# 注册用户
curl -X POST http://localhost:5000/api/v1/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test","email":"test@example.com","password":"123456"}'

安全特性

  • 密码加密 - SHA256哈希不可逆加密
  • API密钥 - 唯一密钥自动生成
  • 唯一性校验 - 邮箱/用户名重复检测
  • 请求限制 - 频率限制(可选)
